maze-solver

This is a square maze solver.

A maze is represented by a matrix like:

[[:S 0 1]
 [1  0 1]
 [1  0 :E]]
  • S : start of the maze
  • E : end of the maze
  • 1 : This is a wall that you cannot pass through
  • 0 : A free space that you can move through.

The goal is the get to the end of the maze. A solved maze will have a :x in the start, the path, and the end of the maze, like this:

[[:x :x 1]
 [1  :x 1]
 [1  :x :x]]

Instructions

  • Clone or fork this repo

  • Install Leiningen

    • Mac OSX

      brew install leiningen

    • UNIX

      wget https://raw.github.com/technomancy/leiningen/stable/bin/lein
  chmod +x lein
  mv lein ~/bin
  lein

  • Run the tests with lein test

  • Prepare the application

    lein uberjar

  • Run the application with a maze

    java -jar target/tiny-maze-0.1.0-SNAPSHOT-standalone.jar "[[:S 0 1][1 0 1][1 0 :E]]"

Note

This application aims to solve the tiny-maze challenge posed by Carin Meier in her wonderland-clojure-katas.
